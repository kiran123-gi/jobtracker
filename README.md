# Job Application Tracker System

## 📌 Overview

The Job Application Tracker System is a web application that helps users manage and track their job applications efficiently.

## 🚀 Features

* Add new job applications
* View all applications
* Update application status
* Delete applications
* Search and filter jobs
* Dashboard with statistics

## 🛠️ Tech Stack

* Frontend: HTML, CSS
* Backend: Django (Python)
* Database: SQL

## 📊 Use Case

This system is useful for job seekers to organize their applications and monitor progress throughout the hiring process.

## ▶️ How to Run

1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run migrations
5. Start server

## 📌 Future Enhancements

* Email reminders
* Authentication system
* Export data to CSV
